# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- GitHub Actions CI pipeline (`ci.yml`) for automated linting, type checking, testing, and building.
- GitHub Actions CD pipeline (`release.yml`) for automated deployment to Vercel upon release.
- Setup test automation with Vitest, Playwright, and React Testing Library.
- Configured PWA support with Service Worker and Manifest caching.
- Created Admin Dashboard with authentication, charts, and product/order management.
- Midtrans Snap API integration (Sandbox) with Webhook signature validation.
- Firebase Firestore and Storage integration with secure rules.
- E-commerce core features: Landing Page, Product Catalog, Cart, and Checkout.
